# spring-boot-ddd-boilerplate

A DDD style boilerplate with spring boot

## tech selection

- Java 8
- Spring Boot 2
- Spring Data Jdbc
- Mysql
- Maven

## principle

- DDD is not goal,the goal is to make architecture be clear.
- Low boilerplate code.
- Necessary layering.
- Easy to use and understand.
- Feature limited.
